                *MOTS DE PASSE APPLICATIFS*

Généralement, pour renforcer la sécurité, les plateformes de messagerie demandent un mot de passe  particulier pour permettre l'accès via des applications tierces. 

Vous devez donc ajouter, à votre compte de messagerie, un mot de passe *applicatif* pour permettre la récupération de vos courriels.

Pour plus d'informations, vous pouvez consulter :
    • Yahoo : [https://fr.aide.yahoo.com/kb/SLN15241.html]
    • GMail : [https://support.google.com/accounts/answer/185833?hl=fr]
    • Hotmail : [https://support.microsoft.com/fr-fr/account-billing/comment-obtenir-et-utiliser-des-mots-de-passe-d-application-5896ed9b-4263-e681-128a-a6f2979a7944]
    • Apple: [https://support.apple.com/fr-fr/102654]
